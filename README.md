# ya-disk-explorer

## Guide
Добавь в /src файл ".env.local" со следующим содержимым (токен в кавычках):
```
VUE_APP_CLIENT_ID=""
```
Чтобы локально запустить сервер с приложением введи команды в корневой папке проекта
``` 
npm install
npm run serve
```
